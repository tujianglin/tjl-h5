# H5 项目框架

## 重要依赖

- `vite3.x`: 打包工具
- `postcss-pxtorem`: px 自动编译转 rem
- `autoprefixer`: css 编译添加 浏览器内核前缀

## 集成

- `vue-router`: 路由
- `axios`: 请求
- `pinia`: 数据存储器
- `pont`: 接口代码生成
- `vant`: UI 库
- `mock`: mock 假数据
- `dingtalk-jsapi`: 钉钉 jsapi

## 钉钉微应用

### 登录案例
